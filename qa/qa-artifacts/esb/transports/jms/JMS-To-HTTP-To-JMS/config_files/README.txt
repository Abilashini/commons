How to use the configuration files
===================================

WSO2 ESB
--------
1. Use the esb-axis2.xml for the WSO2 ESB instance
2. Use the given JMSProxy.xml configuration in your WSO2 ESB instance


WSO2 WSAS/Simple Axis2 Server
-----------------------------
1. Use the backendServer-axis2.xml as the axis2.xml of the backend service you are using. (App server/Simple Axis2 Server)
