** These synapse.xml sample files covers the following VFS transport related scenarios.
Steps to test
==========
Drop the relevant synapse.xml sample file to ESB_HOME/repository/samples folder and start the server with the command ./wso2esb-samples.sh -sn <sample_number>
E.g.:- ./wso2esb-samples.sh -sn 255

The resource files that can be used for testing are located @ https://svn.wso2.org/repos/wso2/trunk/commons/qa/qa-artifacts/esb/transports/vfs/resources
