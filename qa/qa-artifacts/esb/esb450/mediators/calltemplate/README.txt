value_parameters
-------------------------
1. Start WSO2 appserver (offset = 1) and deploy Axis2Service.
2. Use the synapse configuration given in value_parameters.xml
3. Send 'echoInt' requests to http://localhost:8280/services/Proxy1 using Soap UI and monitor the log.
