READ ME
=====================================================================================
1 .Start ESB 

2. Go to the WEB UI  select -> Main -> Connctors ->Add Connectors 

3. Upload a JIRA Connector availabe at the following location and enable the connector 

<ESB><RESPOSITORY><CONNECTORS><JIRA Connector.jar>

4. Create a Proxy and apply any JIRA operation

5. Invoke the proxy using curl -v <end point URL>

6. If JIRA connector artifacts are creating using Developer studio plesse refer the following blog
http://chathurikacm.blogspot.com/2014/01/working-with-esb-jira-connector.html
