Steps
======

a) Deploy the StockPurchasingService.aar available at [1]

b) Add the configuration given in synapse.xml to ESB.

c) Let the task execute and you will see the results at the backend service console as well as ESB console (turn on DEBUG logs)


[1] - https://svn.wso2.org/repos/wso2/trunk/commons/qa/qa-artifacts/app-server/StockPurchasingService.aar
