CMIS Specification Support
============================


1. Download CMIS client from, 
http://www.poolsaboveground.com/apache/chemistry/opencmis/0.10.0/chemistry-opencmis-workbench-0.10.0-full.zip 

2. Extract it. 


3. Run workbench.sh in linux or workbench.bat in windows. sh workbench.sh

4. Then follow th steps given in http://docs.wso2.org/wiki/display/Governance460/Integrate+CMIS+Client+with+WSO2+Governance+Registry
