Steps to reproduce
==================
1. Create a database through Storage Server
2. Then once you get the database URL, edit the NewStratosDBAccessServlet and build the class
3. Copy the built source inside WEB-INF/classes folder
4. Create the war file and deploy
5. Access the webapp from the URL - http://appserver.stratosdev.private.wso2.com/t/[tenantID].com/webapps/NewStratosDBAccessServlet/form.html and hit 'Submit'
