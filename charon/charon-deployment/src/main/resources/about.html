About Charon

WSO2 Charon is the implementation of Simple Cloud Identity Management specification which is provided under Apache 2.0 License.

This be integrated to WSO2 Identity Server to support standardized identity provisioning.

For more details about WSO2 Charon, please refer About Charon.

This webapp represents the reference implementation of SCIM service provider which is shipped with Charon.

This is specifically hosted here to provide a public endpoint for the 1st SCIM Interop Event which is taking place in parallel to IETF 83rd meeting.

To get started, please refer: Getting started with WSO2 Charon reference implementation.

To create a tenant, please follow the link: Register.
