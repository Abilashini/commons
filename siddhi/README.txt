================================================================================
                        Siddhi Complex Event Processing Engine 0.4.0
================================================================================

Welcome to the Siddhi CEP 0.4.0 release

Siddhi CEP is a lightweight and easy-to-use Open Source Complex Event Processing
Engine (CEP) under the Apache Software License v2.0. Siddhi CEP processes
events triggered by various event sources and notify appropriate complex events
according to user specified conditions queries.

This is part of the research project initiated at University of Moratuwa, Sri Lanka,
and further improved by WSO2 Inc.

Key Features of Siddhi CEP
==================================

* Filtering
* Join
* Group By
* Having
* Temporal Windows
    * Time window
    * Time batch window
    * Length window
    * Length batch window
* Pattern
    * Negation
* Sequence
    * Star operation
* Siddhi Query Language - NEW in current release


System Requirements
==================================

1. Minimum memory - 1 GB
2. Processor      - Pentium 800MHz or equivalent at minimum
3. Java SE Development Kit 1.6.0_21 or higher
4. To build Siddi CEP from the Source distribution, it is necessary that you have
   JDK 1.6 or higher version and Maven 2.1.0 or later
