WSO2 UserManager SNAPSHOT Release Notes

[Download] | [Documentation Home] | [Release Note]

23 January, 2007

UserManager is a library allows user management to be consistent across applications.

Key Features of this Release

  • Create/manage/authenticate/authorize users
    • Add/Edit/Delete user
    • Add/Edit/Delete user details
    • Add/Edit/Delete roles
    • Assoicite users to roles
    • Remove users from role
    • Different querries based on users/roles and user information
    • Authenticate users
    • Access control adminstration on users/roles to access resources
    • Access control querries on users/roles
  • Authenticate users in JDBC, LDAP and Acegi user stores
  • Allows authorization based user management
  • All Realms uses the same API, therefore users can be managed across realms without code change.
